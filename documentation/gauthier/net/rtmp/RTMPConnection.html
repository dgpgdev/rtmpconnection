autoConnectproperty 
autoConnect:Boolean  [read-only]

Definit si l'on doit se reconnecter des que l'on detect une deconnection. ATTENTION :si vous désirez couper une connection volontairement, alors que la connexion est en autoConnect, vous devez appeler la methode closeByAction lors d'un appel a la methode close() directement. Sinon votre connection tentera de se reconnecter a nouveau.

The default value is false.

addClientParam()method
public function addClientParam(... parameters):void

Ajoute un ou plusieurs parametre a la connection.

Method qui ajoute un ou plusieurs parametres qui seront envoyé lors de la connection au flash media server ce ci remplace l'ancienne methode qui consisté a faire : var nc:NetConnection = new NetConnection(); nc.connect("rtmp://localhost/test/",param 1,param 2,param 3,...);

Parameters

... parameters — un ou plusieurs parametres


Example 
		 var fmsc:RTMPConnection= new RTMPConnection();
		 fmsc.addClientParam("what else",{login:"toto", mdp:"tata"},512);
		 fmsc.addClientParam("helloworld");
		 fmsc.connection("test");

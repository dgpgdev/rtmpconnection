applicationproperty
public var application:Object

un objet application qui contient la propriété message si elle est definit dans le script ssas lors d'un rejet de connection Il est preferable de recuperer le message par le biais de la propriété rejectReason.


Example
Coté server (fichier .asc) 
		 var error = new Object();
		 error.message = errorID;
		 application.rejectConnection(client, error);
Coté client 
		 nc= new RTMPConnection();
		 nc.addEventListener(RTMPEvent.REJECTED, onRejected);
		 
		 
		 function onRejected(e:RTMPEvent){
		 trace(e.info.rejectReason)
		 }
